   # Installation Guide

   ## Requirements
   - MetaTrader 5 platform
   - Windows 10+ or VPS
   - Minimum $500 trading account

   ## Steps
   1. Download QuantumEdgeProPlus.ex5
   2. Copy to MT5/MQL5/Experts/ folder
   3. Restart MetaTrader 5
   4. Drag EA to chart
   5. Configure parameters
   6. Enable auto-trading
